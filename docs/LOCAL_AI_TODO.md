## Актуальные задачи

### 1. Вынести timezone в системные настройки

Контекст:
- На странице бекапов раньше был текст про `Europe/Moscow`.
- Текст убран из UI, но полноценного поля timezone в настройках пока нет.

Что сделать:
- Добавить в системные настройки поле `Timezone`.
- Сохранять значение в `Setting` с ключом `notification_timezone`.
- Валидировать через `zoneinfo.ZoneInfo`.
- При некорректном значении возвращать понятную ошибку или fallback на `Europe/Moscow`.
- Использовать это значение для времени автобекапов и отображения времени в истории бекапов.

Ожидаемый результат:
- В настройках можно выбрать/указать timezone.
- Страница бекапов не содержит длинных пояснений про timezone.
- Время в истории бекапов форматируется по сохранённой timezone.
