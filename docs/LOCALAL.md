# LOCALAL

Актуальная очередь для локального ИИ.

## Сейчас осталось

1. Добавить поле timezone в настройки приложения.
2. Сохранять значение в `settings.notification_timezone`.
3. Использовать сохранённое значение для отображения времени на странице бекапов.

## Уже сделано в этой итерации

- Порядок вкладок приведён к нужному виду.
- Стрелка в выпадающих списках сдвинута левее.
- В блоке настроек бекапа увеличены отступы и читаемость.
